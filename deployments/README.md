# Deployment Records

This directory contains deployment metadata for InvoiceNFT contracts across different networks.

## Structure

Each deployment creates a JSON file with the following information:
- Network (Base Sepolia / Base Mainnet)
- Proxy address
- Implementation address
- Deployer address
- Block number
- Transaction hash
- Timestamp
