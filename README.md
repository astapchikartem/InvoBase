# InvoBase

NFT-based invoice system on Base L2 with UUPS upgradeable proxies.

## Features

- ERC-721 Invoice NFTs
- Multi-token payment support
- UUPS upgradeable contracts
- Base Pay integration ready

## Tech Stack

- Solidity 0.8.24
- Hardhat
- OpenZeppelin Upgradeable Contracts
- Base Mainnet & Sepolia Testnet

## Setup

```bash
yarn install
yarn compile
yarn test
```

## Deploy

```bash
yarn deploy:sepolia
yarn deploy:mainnet
```
